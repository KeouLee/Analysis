## Analysis
Code reads MD trajectories in DCD (CP2K) or XYZ format and then calculates the free energy curve/surface, reorientation
frequency, and the time-correlation function of angular velocity.

## conductivity
The computed mean square dipole displacements of six solid electrolytes, along with the corresponding scripts.

## defect
The VASP input files for point defect calculations for Na3OBH4, using a sodium vacancy with a charged of -2 as an example.

## metadynamics
The CP2K input files for well-tempered metadynamics simulations of LiBH4, together with the post-processing scripts.
